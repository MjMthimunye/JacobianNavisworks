# 🧩 Jacobian Navisworks

A custom Autodesk Navisworks add-in focused on improving productivity, automation, and workflow efficiency within BIM coordination environments.

This project is part of the **Jacobian Dev** ecosystem — building practical tools for real-world engineering and construction workflows.

---

## 🚀 Features

- 🔧 Custom Navisworks plugin tools
- ⚡ Workflow automation for repetitive tasks
- 🧠 Improved coordination and model navigation
- 📊 Enhanced data handling inside Navisworks
- 🏗️ Built for BIM, MEP, and coordination use cases

---

## 🛠️ Tech Stack

- C# (.NET Framework)
- Autodesk Navisworks API
- WPF / WinForms (UI)
- MVVM (where applicable)

---

## 📦 Installation

1. Build the solution in **Release mode**
2. Locate the output `.dll` file
3. Copy the add-in files to your Navisworks plugins directory:

```
C:\Program Files\Autodesk\Navisworks Manage <VERSION>\Plugins\
```

4. Restart Navisworks

---

## ▶️ Usage

Once installed:

- Open Navisworks
- Navigate to the **Add-ins / Plugins** tab
- Launch the Jacobian tools from the ribbon or plugin panel
